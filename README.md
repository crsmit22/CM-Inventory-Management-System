Welcome to the Campus Ministry Sacristy Inventory Management System!

This system is used to keep careful track of what we store in our sacristy here on campus in Saint Thomas More Chapel.
Please double check everything you add to the inventory and be certain that you keep your new additions organized properly. 
Also ensure that your item is not already listed.

To run this system, launch the files to VS Code and go to the "app.py". In the terminal, install Flask by typing 'python -m pip install flask'
When installed, run "app.py' then the terminal will show your locally hosted server.

With your help, we can maintain a detailed and up-to-date inventory of our sacristy.
Thank you for your help!
